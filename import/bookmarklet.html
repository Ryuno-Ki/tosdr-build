Google Groups thread subjects scraper

This is aimed at scraping the subjects of ToS;DR threads, but you could adapt it to other forums by changing the string 'tosdr' in the bookmarklet code to something else.

  • Drag the following link: ToS;DR forum scraper from here to your bookmark bar
  • Visit the ToS;DR forum
  • Wait for the topics to load
  • Scroll down, so that more topics load (depending how long ago you scraped for the last time)
  • Now click the bookmarklet in your browser's bookmark bar
  • The page will turn into a JSON document which maps topic hash to topic subject
